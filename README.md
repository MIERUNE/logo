## MIERUNEロゴマークについて

MIERUNEは、様々なデータを「見える」かたちにするというミッションを持っています。このミッションをもとに、[IMPROVIDE](https://improvide.co.jp/)さんによって、このロゴマークが生まれました。

ロゴは、漢字の「目」をモチーフとし「見る」ことを表しています。また、「目」だけでなく、「M」や「E」といった多方面からの解釈も想起させるデザインになっています。
